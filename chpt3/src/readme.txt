Beginning Groovy and Grails Chapter 3 Code

Chapter 3 demonstrates some of the advanced features of Groovy
like unit testing, XML processing, templating, Expandos, 
Meta Object programming and Domain-specific languages.

Unit testing

To run the Groovy unit test example type the following:

>groovy RangeTest.groovy

XML Processing

To run the Java example of writting XML type the following:

>javac GenerateXML.java
>java GenerateXML

To run the Groovy example of writting XML type the following:

>groovy GroovyGenerateXML.groovy

Templating

To run the Groovy templating example to generate an email message type the following:

>groovy TemplateExample.groovy

Expandos

To run the simple Expando example type the following:

>groovy ExpandoExample.groovy

To run the templating example using Expandos type the following:

>groovy ExpandoTemplateExample.groovy

Meta Object Programming

To run the templating example which uses meta objects type the following:

>groovy MOPTemplateExample.groovy

To run the meta object programming examples for loading files type the following:

>groovy MetaProgrammingExample.groovy

Domain Specific Language

To run the Groovy domain specific language type the following:

>groovy DomainSpecificLanguageExample.groovy
